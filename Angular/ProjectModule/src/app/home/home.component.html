Project Code Module:


Following is a list of functionalities of the system. There is an admin who can add, delete, update and view thng project details.

1. Add Project Code details:

•Insert the values into database table if data is valid, else display appropriate error messages.

•Project Code has to be auto generated.

•Project Type must start with capital character, should only contain alphabet and it should contain minimum 3 and maximum 15 characters (cannot be empty).

•Based on existing project code, update the following fields Project Code, Project Type and Project Description

•Project Description can be a combination of alphabets, digits and underscores and it should contain a minimum of 15 and a maximum of 100 characters (cannot be empty).

2. Modify Project details:

•Based on existing project code, update the following fields Project Code, Project Type and Project Description (validation should be taken care of).

3. Display Project details:

•Based on existing project code, display the following fields Project Code, Project Type and Project Description.

4. Display All Project details:

•Display the following fields Project Code, Project Type and Project Description for ang project codes in the database.

5. Delete Project details:

•Based on existing project code, delete the following fields, Project Code, Project Type and Project Description.
